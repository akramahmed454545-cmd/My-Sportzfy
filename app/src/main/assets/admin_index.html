OBS Studio Live Broadcaster Receiver

Directly receive and stream from OBS Studio to the Android app!

Offline
Playlist Location in App:
/obs/live/index.m3u8

🛡️ Step A: In OBS Studio, go to Settings ➔ Stream.

⚙️ Step B: Set Service to HLS (do NOT use 'Custom...').

🔗 Step C: Choose Custom Server and paste this exact Server URL:

http://localhost:8080/obs/live/

🔑 Step D: Keep Stream Key completely blank or type any value, then save.

📡 Step E: Click Start Streaming in OBS. The Android app automatically provides a premium Local OBS Broadcast player card on the home screen to watch the feed instantly!
